# [Any (L4D/L4D2)] Fix Exec Config Unicode

### Introduction
- Fix **quoted** unicode characters being ignored when execing configs.
  - It's possible now to write something directly like  `hostname "提瓦特大陆"` in your "server.cfg".
- (Development) Currently only L4D/L4D2 are supported. Feel free for request for other Source games with this issue.

<hr>

### Requirement
- [Source Scramble](https://forums.alliedmods.net/showthread.php?t=317175)

<hr>

### Installation
1. Put **plugins/fix_exec_config_unicode.smx** to your _plugins_ folder.
2. Put **gamedata/fix_exec_config_unicode.txt** to your _gamedata_ folder.
