# [L4D2] Fix Mixed Characters

### Introduction
- Prioritize character searching (Team 4 bots) to hopefully fix issues with 8+ survivors.
  1. Fix issues with L4D1 bots on maps like The Passing.
    - Players using L4D1 character may block L4D1 bots from spawning and mess with the process.
  2. Workaround on characters across different survivor sets to make intro cutscenes work on them.
- More fix cases are yet not discovered and reports are welcomed.

<hr>

### Requirement
- **DHooks**

<hr>

### Installation
1. Put **plugins/l4d2_fix_character_mixed.smx** to your _plugins_ folder.
2. Put **gamedata/l4d2_fix_character_mixed.txt** to your _gamedata_ folder.
