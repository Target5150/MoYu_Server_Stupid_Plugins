# [L4D & 2] Transition Entity

### Introduction
- Provide events for weapons/spawners/props transition across level change.
  - See __scripting/include/l4d_transition_entity.inc__ for detail.

<hr>

### Requirement
- **DHooks**

<hr>

### Installation
1. Put **plugins/l4d_transition_entity.smx** to your _plugins_ folder.
2. Put **gamedata/l4d_transition_entity.txt** to your _gamedata_ folder.
