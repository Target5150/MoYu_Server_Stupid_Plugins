# [L4D & 2] No Drag Shove

### Introduction
- Disallow shoving since the moment being pulled while shooting is allowed.
	- Just like when melee pulled out.

<hr>

### Installation
1. Put the **l4d_no_drag_shove.smx** to your _plugins_ folder.
