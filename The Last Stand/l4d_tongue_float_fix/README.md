# Tongue Float Fix

### Introduction
- Fix tongue instant choking survivors.
	- Usually happens when as Survivor grabbed by tongue in the air or standing on some entities.

<hr>

### Installation
1. Put **plugins/l4d_tongue_float_fix.smx** to your _plugins_ folder.
